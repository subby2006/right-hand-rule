# right-hand-rule

A SolidJS application demonstrating the right hand rule for electromagnetism.

Users may enter two of the three variables (magnetic field `b`, velocity `v`,
or force `f`) and the app computes the missing value. An interactive Three.js
canvas visualizes the relationship.

## Get Started

1. Navigate into the project directory:
   ```bash
   cd right-hand-rule
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:5173` (or another port if
5173 is in use).

## Commands

| Command | Description |
| ------- | ----------- |
| `npm run dev` | start local dev server |
| `npm run build` | build for production |
| `npm run preview` | locally preview production build |
